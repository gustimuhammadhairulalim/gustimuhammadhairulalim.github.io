// 01. Tentang

Profil Saya

Saya merupakan mahasiswa Semester 6 Program Studi Teknologi Rekayasa Elektronika, Jurusan Teknik Elektro di Politeknik Negeri Semarang, dengan latar belakang pendidikan Sekolah Menengah Kejuruan pada Jurusan Teknik Jaringan Tenaga Listrik di SMKN 2 Pangkalan Bun. Memiliki minat dan fokus pada bidang otomasi industri, sistem kontrol, instrumentasi, serta integrasi perangkat keras dan perangkat lunak dalam sistem elektronika terapan. Saya lebih berorientasi pada pembelajaran berbasis praktik (hands-on) dan pengembangan sistem secara langsung di lapangan, khususnya dalam proses perancangan, perakitan, pengawatan, hingga troubleshooting dalam sebuah sistem elektronika dan otomasi. Selain itu, saya juga memiliki pengalaman dalam pengembangan proyek berbasis mikrokontroler dan IoT menggunakan Arduino, ESP8266, serta perangkat otomasi lainnya. Saya memiliki ketertarikan untuk terus mengembangkan kompetensi di bidang maintenance, automation system, industrial control, dan teknologi industri berbasis digital. Saat ini sedang mengerjakan Tugas Akhir berupa sistem sorting otomatis berbasis YOLOv8 dan PLC yang menggabungkan teknologi computer vision, kontrol industri, sensor, dan otomasi untuk proses identifikasi dan pemisahan objek secara otomatis.

Karya & Proyek

Instalasi Kelistrikan Residensial
MAGANG

Proyek instalasi kelistrikan rumah bersama PT Karya Mitra Sejahtera, meliputi wiring distribusi daya, pemasangan panel listrik, dan instalasi KWH meter sesuai standar PLN. Hands-on di berbagai titik instalasi riil.

WATER LEVEL CONTROL BERBASIS PID
Tugas Akhir Semester

Merancang sistem kontrol ketinggian air secara otomatis menggunakan Arduino dan sensor VL6180X. Menggunakan metode PID di program menggunakan Matlab Simulink.

Arduino Uno Arduino Nano Matlab Sensor VL6180X
Sistem Sorting Otomatis Berbasis YOLOv8 & PLC
TUGAS AKHIR

Sistem penyortiran objek geometri otomatis yang mengintegrasikan computer vision (YOLOv8) dengan PLC untuk klasifikasi bentuk (lingkaran, segitiga, persegi) sekaligus deteksi cacat seperti bentuk tidak sempurna atau geometri yang hilang. Novelti utama: kemampuan deteksi defect yang belum ditemukan pada penelitian sebelumnya.

YOLOv8 PLC Computer Vision Defect Detection Otomasi Industri Python
